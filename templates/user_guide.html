3. Player Stats Explained

Batsman Quick Stats:
  • Matches: Total matches played
  • Runs: Total runs scored
  • Average: Runs per dismissal (Runs / Times Out)
  • Strike Rate: (Runs / Balls) √ó 100
  • 100s / 50s: Centuries / Fifties scored
  • Highest Score: Top runs in a single match
  • RPI (All/1st/2nd): Runs per innings for all, 1st, and 2nd innings
  • Dot %: % of balls faced scoring zero
  • BPB (Balls Per Boundary): Balls faced / (Fours + Sixes). How often a batsman hits a boundary (lower = better).
  • BPD (Balls Per Dismissal): Balls faced / Times out. How long a batsman survives before getting out (higher = better).
Bowler Quick Stats:
  • Matches: Total matches bowled
  • Wickets: Total wickets taken
  • Avg: Runs conceded per wicket (lower = better)
  • Eco: Economy rate (Runs per over, lower = better)
  • SR: Balls bowled / Wickets. Average balls per wicket (lower = better)
  • Wkts (1st/2nd): Wickets in 1st and 2nd innings
  • Best: Most wickets in a single match
  • 5W+: Times bowler took 5+ wickets in a match
  • Dot %: % of dot balls (no run balls)
  • BPB (Balls Per Boundary Conceded): Balls Bowled / (Fours + Sixes Conceded). How often a bowler is hit for a boundary (higher = better).
